# :deciduous_tree: EasyValetParking
    
  
  
### 개요
호텔에서 일하면서 느꼈던 불편함을 소프트웨어를 통해 자동화하여 개선하고자 프로젝트를 시작합니다.  
매일 좋은 서비스를 위해 수고하시는 동료분들이 조금은 편하길 바랍니다.  
  
  
## 사용 기술 및 개발 환경
Java 11, Mysql, Mybatis, Redis, Spring Boot, Naver Cloud, IntelliJ, github actions
